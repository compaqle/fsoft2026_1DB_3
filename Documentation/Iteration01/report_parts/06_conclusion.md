# Conclusão

Esta primeira iteração (Análise de Requisitos e Modelação) permitiu definir as fronteiras do nosso programa e compreender o que os utilizadores finais precisam. A separação entre as vontades do utilizador final (User Stories) e a especificação técnica (Casos de Uso) fornece uma base sólida para a Iteração seguinte. O próximo passo irá envolver o aprofundamento destas entidades conceptuais para classes em C++, respeitando o padrão MVC.
